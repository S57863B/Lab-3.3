This project involves modifications of two previous Frontend Mentor challenges by replacing custom CSS with Bootstrap 5 utility classes and components.
The goal was to minimize custom CSS. The main challenge that I faced during my work is a massive amount of information in the Bootstrap documentation.
Navigating through various components and utility classes. To efficiently parse the documentation and speed up the development process, I urilized AI tools (Gemini).
This allowed me to better understand Bootstrap's logic and find specific classes for layout and alignment. The transition required changing a large portion of the original code.
I had to restructure the HTML to properly implement Bootstrap's Grid system and Card components.
